# Miscellaneous Scripts  


This folder contains miscellaneous scripts to help maintain this repo.  

## Update README - Skills and Steering  

The README references Skills and Steering files. We do not want to have to manually edit everytime some change is made to either folder. Thus, created skills to read the frontmatter of both and update the README accordingly:  

**For Skills**  

```bash
chmod +x update-skills-references.sh  
./update-skills-references.sh
```  

**For Steering**  

```bash
chmod +x update-steering-references.sh  
./update-steering-references.sh
```
